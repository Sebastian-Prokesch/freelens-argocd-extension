I’ve just released an Argo extension. The first goal was to show the status of ArgoCD Applications, Argo Rollouts, and Argo Workflows. I plan to extend it with more actions for Argo resources, but for now it should work well as a viewer.

https://github.com/Sebastian-Prokesch/freelens-argocd-extension

I’d be happy if some of you could try it out, test it against real clusters, report bugs, suggest improvements, or open PRs. Feedback is very welcome, especially around UX, missing Argo resource details, and common GitOps workflows.

Disclaimer: I’m not a professional developer, so the extension is not perfect and there are probably bugs.
